Manuel — Trouver l’ID d’un calendrier Google (Calendar ID)

KaruZen — synchronisation Supabase / Google Calendar · CALENDAR_ID

1. À quoi sert le Calendar ID ?

  • Chaque agenda Google (principal, professionnel, « RDV site », etc.) possède un identifiant unique.
  • Les applications utilisent cet ID pour savoir dans quel agenda créer ou modifier les événements.
  • Ce n’est pas un mot de passe : c’est un identifiant technique, visible dans les paramètres du calendrier.

2. Avant de commencer

  • Connecte-toi avec le compte Google qui possède l’agenda concerné.
  • Ouvre : https://calendar.google.com

3. Méthode recommandée : Paramètres du calendrier

Étape 1 — Liste des agendas

  1. À gauche, sous le mois, repère la liste « Mes agendas » (My calendars).
  2. Choisis l’agenda dont tu veux l’ID (ex. ton nom, ou un agenda dédié « Rendez-vous »).

Étape 2 — Menu des paramètres

  1. Survole le nom de l’agenda avec la souris.
  2. Clique sur les trois points ⋮ à droite du nom.
  3. Choisis « Paramètres et partage » (Settings and sharing).

Étape 3 — Lire le Calendar ID

  1. Descends jusqu’à la section « Intégrer le calendrier » (Integrate calendar).
  2. Dans le champ « ID du calendrier » (Calendar ID), sélectionne tout le texte et copie (Ctrl+C / Cmd+C).
  3. Colle cette valeur dans le secret Supabase CALENDAR_ID, sans espace avant ou après.

4. Exemples de forme d’ID

Type d’agenda Exemple d’ID
Agenda principal Souvent ton adresse e-mail Gmail complète (ex. prenom.nom@gmail.com)
Cas particulier Parfois le mot primary si l’outil ou la doc le demande explicitement
Agenda secondaire Souvent une chaîne du type xxxx@group.calendar.google.com

5. Partage avec le compte de service (rappel)

Pour que la synchronisation fonctionne, le même agenda doit être partagé avec l’e-mail du compte de service Google Cloud (se terminant par @….iam.gserviceaccount.com), avec le droit Modifier les événements. Détail : fichier docs/GOOGLE_CALENDAR.md dans le projet.

6. Problèmes fréquents

Problème Piste
Section « Intégrer le calendrier » introuvable Descends plus bas dans la page Paramètres ; le libellé peut varier selon la langue.
Plusieurs agendas Chaque calendrier a son propre ID : répète les étapes pour celui utilisé pour les RDV.
Mauvais compte Vérifie l’avatar / compte en haut à droite sur calendar.google.com.
